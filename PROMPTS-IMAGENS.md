# MGX Gaming — Prompts de Imagens

## Guia de Estilo (lê antes de gerar)

**Paleta obrigatória em TODAS as imagens:**
- Fundo: Preto profundo `#0e0e0e`
- Superfícies: Cinza muito escuro `#131313` / `#1f1f1f`
- Acento ÚNICO: Amarelo dourado `#FFE792` / `#FFD709`
- SEM azul, SEM vermelho, SEM roxo, SEM verde

**Estética geral:** Esports urbano africano · Maputo, Moçambique · Cinematográfico · Alta intensidade · Minimalista brutal

**Logo:** Tens o logo MGX Gaming — coloca-o em TUDO que for peça de marca (roupa, periféricos, posters). Quando o prompt diz `(Logo-Mgx.jpeg — círculo preto com "MGX GAMING" em dourado com raios elétricos)`, substitui pelo logo que tens.

## NOTAS DE USO

**Ao gerar com GPT Image:**
- Coloca o logo MGX no chat antes de enviar o prompt
- Se a IA não colocar o logo automaticamente, pede: *"Add the MGX logo I provided to [location] in golden yellow"*
- Para textos específicos no poster, confirma depois de gerar: *"The text [X] should read exactly: [texto]"*

**Ao gerar com NanoBanana Pro:**
- Faz upload do logo antes de iniciar
- Usa as dimensões exactas: 1920×1080 para banners web, 1080×1080 para posts, 1080×1920 para stories
- Se o resultado tiver cores erradas (azul, vermelho), adiciona: *"Remover completamente qualquer cor que não seja preto ou amarelo dourado #FFE792"*

**Guardar as imagens:**
- Banner Homepage Hero → `public/hero-homepage.jpg`
- Card FC25 → `public/fc25-card.jpg`
- Card Fortnite → `public/fortnite-card.jpg` (substitui o actual se melhor)
- Banner FC25 torneio → `public/fc25-torneio.jpg`
- Banner Fortnite torneio → `public/fortnite-torneio.jpg` (substitui o actual se melhor)
- Depois actualiza os `src=` nas respectivas páginas
